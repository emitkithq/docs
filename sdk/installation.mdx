## Installation

<CodeGroup>

```bash npm
npm install @emitkit/js
```

```bash pnpm
pnpm add @emitkit/js
```

```bash yarn
yarn add @emitkit/js
```

</CodeGroup>

## Requirements

- Node.js 16+
- TypeScript 4.5+ (optional)

## Basic Setup

```typescript
import { EmitKit } from '@emitkit/js';

const client = new EmitKit('emitkit_xxxxxxxxxxxxxxxxxxxxx');
```

See [Quick Start](/quickstart) for a complete guide.
